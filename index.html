Network Intrusion Detection using Autoencoders

Flight Fare Predition and Analytics

This project focuses on predicting flight fares using a comprehensive dataset from Kaggle. The process involves extensive data preprocessing, feature engineering, and the application of multiple machine learning algorithms to achieve accurate fare predictions. The project demonstrates a solid understanding of data analysis, feature engineering, and machine learning model development. This project not only highlights technical proficiency in data science and machine learning but also provides valuable insights into the factors influencing flight ticket pricing.

Exploratory Data Analysis on Tweet Data

This comprehensive project involves analyzing a dataset of 10,000 tweets from January 2020. The analysis encompasses various aspects of the tweets, such as the sources used to post them, the number of replied and quoted tweets, user statistics, sentiment analysis, word cloud generation, and more. Additionally, the project includes sentiment analysis to segregate COVID-19 related tweets from non-COVID-19 tweets, providing insights into public sentiment during the early stages of the pandemic.

The findings can be used to understand trends and patterns in Twitter usage during January 2020, particularly in relation to the early stages of the COVID-19 pandemic. The project demonstrates proficiency in data analysis, sentiment analysis, and data visualization using Python.

View this project on GitHub : EDA-on-Tweet-Data

Tour planning using Kruskal's Algorithm

Aim of the project is to visualize UNESCO World Heritage Sites within a specified country and compute a Minimum Spanning Tree (MST) to determine the optimal travel route to visit all these sites. The MST is calculated using Kruskal's algorithm, ensuring minimal travel distance between the sites.

This project effectively demonstrates the use of data processing, algorithm implementation, and visualization techniques to analyze and optimize travel routes for UNESCO World Heritage Sites within a specified country. The approach ensures minimal travel distance, making it a valuable tool for travelers and researchers alike.

View this project on GitHub : Tour-Planner-using-Kruskal-s-Algorithm

Car Customization Application using Augmented Reality

The Car Customization Application is an advanced AR tool designed to enhance car customization using ARFoundation in Unity. Leveraging the ARCore XR Plugin for Android, it offers a rich set of AR functionalities. The application uses the device's camera to provide a live feed, acting as the AR camera. It performs plane detection to create GameObjects for each detected plane, allowing markerless placement of virtual car parts or accessories. Users can place, scale, and rotate virtual objects in the real environment. The rendering component combines the live camera feed with virtual objects, providing a realistic preview of the customized car. This application showcases advanced AR capabilities, enabling users to visualize and interact with car customizations in their real environment, highlighting expertise in AR development and practical applications of AR technology to enhance user engagement and satisfaction.

View this project on GitHub : Car-Customization-Application-using-Augmented-Reality
